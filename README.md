**(C# 코딩) 심플 계산기
개요
C# 프로그래밍 학습
1줄 소개: 사용자 입력을 통해 사칙연산을 수행하는 계산기 프로그램
사용한 플랫폼:
C#, .NET Windows Forms, Visual Studio, GitHub
사용한 컨트롤
TextBox (입력값, 결과값 표시)
Button (숫자, 연산, 기능 버튼)
ListBox (계산 기록)
사용한 기술과 구현한 기능
Visual Studio를 이용한 UI 디자인
Button Click 이벤트를 이용한 기능 구현
string 데이터를 double로 변환하여 계산 처리
조건문(if)을 이용한 사칙연산 분기 처리
예외 처리 (0으로 나누기 방지, 입력값 검증)
Substring을 이용한 문자 삭제 기능 구현
수업 중 배운 내용 적용
TextBox를 이용한 사용자 입력 처리
Button 이벤트를 활용한 프로그램 흐름 제어
변수(op, isSecond)를 이용한 상태 관리
ListBox를 이용한 계산 기록 저장 및 출력
실습 중 구현한 기능들 설명
숫자 버튼 클릭 시 TextBox에 값 입력
연산 버튼 클릭 시 연산자 저장
'=' 버튼 클릭 시 결과 계산 및 출력
C 버튼: 전체 초기화 기능
CE 버튼: 현재 입력값 전체 삭제
Del 버튼: 마지막 입력값 한 글자 삭제
계산 결과를 ListBox에 기록
계산 후 결과를 다음 계산에 활용 가능
소수점(double) 계산 지원
잘못된 입력을 방지하기 위한 예외 처리 구현**
