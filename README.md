\## 실행 화면 (과제1) ![과제1 실행화면](img/https://github.com/njsin/SimpleCalculator/raw/master/img/screenshot-1.png)



\### 과제 내용



\* TextBox와 Button을 이용하여 숫자 입력 UI를 구성합니다.

\* 숫자 버튼 클릭 시 TextBox에 값이 입력되도록 구현합니다.

\* '+' 버튼과 '=' 버튼을 이용하여 두 수의 덧셈을 수행합니다.

\* 계산 결과를 TextBox에 출력합니다.



\### 구현 내용 및 기능 설명



\* 숫자 버튼 클릭 시 입력값이 TextBox에 누적되어 표시된다.

\* '+' 버튼 클릭 시 두 번째 숫자를 입력받기 위한 상태로 전환된다.

\* '=' 버튼 클릭 시 두 숫자의 덧셈 결과가 출력된다.



\## 실행 화면 (과제2)![과제2 실행화면](img/https://github.com/njsin/SimpleCalculator/raw/master/img/screenshot-2.png)



\### 과제 내용



\* 사칙연산(+,-,\*,/) 기능을 추가합니다.

\* 각 연산 버튼 클릭 시 연산자를 저장합니다.



\### 구현 내용 및 기능 설명



\* '+' , '-' , '\*' , '/' 연산 수행 가능

\* 연산자(op)를 저장하여 '=' 버튼에서 계산 처리

\* 조건문을 이용하여 연산 분기 처리
